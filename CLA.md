## How to sign

When you open a pull request, the CLA Assistant bot will check whether you have
signed. If you have not, it will comment on your pull request. To sign, add a
pull request comment with exactly the following text:

```
I have read the CLA Document and I hereby sign the CLA
```

Your GitHub username and the signing event are then recorded in this
repository's signature file. You only need to sign once.
